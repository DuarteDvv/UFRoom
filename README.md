# UFRoom

## Objetivos e Principais Features

Nosso objetivo é a construção de uma plataforma de anúncios de aluguel focado em estudantes, restrito a repúblicas e kitnets.

Funcionalidades no escopo do anunciante: 
- Cadastro e autenticação
- Opção de cadastrar um imóvel com imagens e descrição
- Opção de editar ou remover os imóveis

Funcionalidades no escopo do usuário que busca uma acomodação:
- Filtrar por localização (ou Universidade) e especificações de acomodação, sexo, vagas disponíveis
- Acessar um catálogo de anúncios
- Buscar por palavra-chave
- Acessar as informações do imóvel
- Acessar o contato do anunciante
